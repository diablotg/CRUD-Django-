# Django CRUD Application

Aplicación web desarrollada con Django que implementa operaciones básicas CRUD (Create, Read, Update, Delete).

## 🚀 Características

- Gestión de registros (crear, listar, actualizar y eliminar)
- Arquitectura basada en Django (models, views, templates, forms)
- Integración con base de datos PostgreSQL (Supabase)
- Uso de variables de entorno para configuración sensible

## 🛠️ Tecnologías

- Python 3.x
- Django
- PostgreSQL
- Supabase

## 📦 Instalación

- pip install -r requirements.txt
- pip install django psycopg2-binary python-dotenv

## Archivo .env

DB_NAME=your_db_name
DB_USER=your_user
DB_PASSWORD=your_password
DB_HOST=your_host
DB_PORT=5432
